B | Übung 1

1-Faktor vs. 3-Faktor

WichtigAufgabe
  1. Spezifizieren Sie ein 1-Faktor-Modell (alle 9 Items auf einen Faktor g)
  2. Schätzen Sie es mit cfa()
  3. Vergleichen Sie den Fit mit dem 3-Faktor-Modell (Tabelle)
  4. Führen Sie einen χ²-Differenztest durch: anova(fit_1f, fit_3f)
  5. Diskutieren Sie: Ist der 3-Faktor-Fit signifikant besser? Wie groß ist die Verbesserung in CFI?
Daten
Hilfestellung

Die Items heißen x1, x2, … , x9 im Datensatz. Kopervorlagen für das Modell finden Sie in folgendem R-Block, der nicht läuft:

# Item-Namen für 1-Faktor Modell:
# x1 + x2 + x3 + x4 + x5 + x6 + x7 + x8 + x9

# Item-Namen für 3-Faktor Modell (jede Zeile ein Faktor)
# x1 + x2 + x3 
# x4 + x5 + x6 
# x7 + x8 + x9

Weitere Hilfestellung:

# Der Aufruf von lavaan erfolgt über 

VAR_NAME <- cfa(model, daten)

# Die Daten sind in folgendem R-data.frame:

head(HolzingerSwineford1939)

# Um den Fit zu vergleichen nutzen Sie die Funktion

fitMeasures(fit_1f, c("chisq", "df", "cfi", "rmsea", "srmr"))
fitMeasures(fit_3f, c("chisq", "df", "cfi", "rmsea", "srmr"))

# fit_1f ist dabei die Variable, in der das Ergebnis
#        des Einfaktor-Modells erwartet wird
# fit_3f ist dabei die Variable, in der das Ergebnis
#        des Dreifaktor-Modells erwartet wird

# Für den Modellvergleich nutzen Sie die Funktion 

anova(fit_1f, fit_3f)

# Wenn Sie sich ein Pfaddiagramm ausgeben 
# lassen wollen, geht das mit folgenden R-Befehlen:

library(lavaanPlot)
lavaanPlot(model = fit_3f,   
     coefs = TRUE, stand = TRUE,covs = TRUE)
